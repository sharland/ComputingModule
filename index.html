KS3 Computing Module Template

A simple structure for developing and sharing KS3 Computing modules

About this Module template

This template is designed to enable teachers to create their own modules for KS3 Computing, share them and then use modules developed by others. This is about giving you a framework to continue developing your own work for your school but know that it is usable and shareable with others and you can incorporate what others do into your own curriculum.

How does it work?

The structure is designed to be as simple as possible and currently relies on md files. You will therefore need to know a little bit about Markdown (but it's not hard!).

Resources and Links

If you are new to teaching Computing and would like some help getting started with writing a module or working with an existing one the following resources may be of use to you.

Using Github to help build your curriculum

Github is a web-based hosting service for code projects stored in repositories. It's used by developers to store open-source work which can then be easily tracked, shared and collaborated on with many others. As an example of good industry practice it is a great way to get pupils involved with the open-source development community. It also makes sense therefore to develop your own resources using Github.

Writing with Markdown

Markdown is a syntax for writing text documentation that can then be converted into simple HTML. Github uses its own version of markdown which has some slight differences with normal markdown. Markdown is used frequently on Github for creating readme files and is therefore ideally suited for creating documents which can be used on Github.

The Computing at School Community

If you are looking for a community of like-minded educators who are grappling with (and winning!) issues to do with teaching Computing in schools then this is the best place to go. With many online resources and forums as well as real-world training days, conferences and the famous CasHub you will not lack for ideas, inspiration and best of all a chance to share what you have done yourself.

Collecting and sharing with Delicious.com

Delicious.com is a social bookmarking website for collecting links from the Internet. It works very well as a tool for gathering favourites from twitter and then once you have manually tagged them, sharing those links becomes very easy. As a tool for tracking online resources it is absolutely essential.
